# Monkey Script for Payment

A userscript to add a custom form on the payment page.

## Installation

Click here to install: [Install Script](tampermonkey://install?url=https://raw.githubusercontent.com/sacrosanctic/monkeyscripts/main/monkeyscript.js)

Requires Tampermonkey extension.

## Testing

To test the script locally:

1. Install dependencies: `pnpm install`
2. Run the test server: `pnpm serve`
3. Open http://localhost:5173 in your browser.
4. The script should add a form to the blank page.
